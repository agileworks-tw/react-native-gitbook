# 實戰 ToDo List App

運用 View, Text, TextInput, Button, FlatList 組件與 state 更新概念，做出一個可以新增刪除項目的 ToDo List App

1. FlatList 加上列表資料來源
2. 將 TextInput 改成 controlled component
3. 完成新增 ToDo 項目功能
4. 完成 ToDo 項目的刪除功能
